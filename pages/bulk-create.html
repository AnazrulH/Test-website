Canva App – Bulk Create

Langkah asas membuat banyak reka bentuk menggunakan data dari Excel/Google Sheets. :contentReference[oaicite:3]{index=3}

  1. Sediakan data (Excel / Google Sheet) — contoh: Nama, Kelas, Tajuk.
  2. Buka templat reka bentuk di Canva (contoh: sijil A4).
  3. Menu: Apps → Bulk Create → Upload data (Excel / CSV).
  4. Pilih elemen teks untuk di-link ke lajur data dan tekan “Create”.
  5. Simpan dan semak fail yang terhasil (contoh: “(Bulk 1) Sijil Penghargaan”).

Cara menyesuaikan

Anda boleh sambungkan lebih satu elemen teks, gunakan gambar dinamik (jika ada URL), dan muat turun semua hasil serentak.

Menjana sijil penyertaan pelajar, tag nama acara, atau label program secara automatik.
