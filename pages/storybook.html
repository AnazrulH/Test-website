Storybook: Gemini & ChatGPT + Canva

Menggunakan Gemini (Google) atau ChatGPT untuk menjana teks cerita dan Canva untuk layout gambar. :contentReference[oaicite:6]{index=6}

Workflow ringkas

  1. Gunakan ChatGPT / Gemini untuk tulis teks cerita.
  2. Hasilkan image prompts (modul memberi contoh format image prompt dalam Bahasa Inggeris).
  3. Di Canva: pilih Document A4 → Apps → Magic Media → Paste prompt + Seed → Generate.
  4. Set image as background atau place image → tambah teks kisah → export sebagai PDF atau web page.

Image prompts mesti: 1) gambaran watak utama konsisten, 2) aksi & persekitaran berubah ikut halaman, 3) gunakan gaya watercolor/storybook.

Example prompt (from module)

A cheerful golden brown dog with bright gentle eyes... playing together in a sunlit garden..., watercolor storybook style
